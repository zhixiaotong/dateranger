# dateranger
基于momentjs、element-ui风格的vue日期拖拽选择插件

## 在线预览：
https://zhixiaotong.github.io/demo-online/example/dateranger

## 安装项目依赖
```
npm install
```

### 本地开发者模式运行
```
npm run serve
```

### 线上模式打包运行
```
npm run build
```

### 执行测试
```
npm run test
```

### 提示和修复文件
```
npm run lint
```

### 自定义配置
请参考 [Configuration Reference](https://cli.vuejs.org/config/).
